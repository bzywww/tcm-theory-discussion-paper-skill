# Style Rules

## Tone

Write as a Chinese academic TCM theory discussion article. Use direct claims, clear transitions, and compact paragraphs. Avoid chatty explanations.

Prefer:

- “其机在于……”
- “其本在……”
- “其变在……”
- “治当……”
- “此为……之枢”
- “临证宜……”

Avoid:

- “本文认为”
- “由此可见”
- “该理论”
- “我们可以理解为”
- “二者虽分属不同体系”
- “既往《中医学报》相关研究”
- “例如/譬如” when used as casual exposition

## Headings

Headings should form an argument chain. A reader should understand the paper's logic by reading headings alone.

Good heading chain shape:

1. [clinical state]为[stage/problem]之候
2. [classic concept]为立论之本
3. [memory/pathogen/qi-blood link]为发病之机
4. [transmission]为迁延之枢
5. [variants]为证候分化之纲
6. [treatment principles]为辨治之要

## Modern Mechanism Language

Name only the concepts needed for the argument. Define them before abbreviating or relying on them.

For biomedical mechanisms, use concise explanatory phrases and quickly return to the TCM pathogenesis chain.

## Treatment Language

Do not list formulas as decorative evidence. For each formula, explain:

1. applicable pattern;
2. treatment method;
3. formula meaning;
4. return to core pathogenesis.

## Final Cleanup

Remove drafting markers, comments, bracketed reminders, internal notes, and uncertainty flags before delivery.
